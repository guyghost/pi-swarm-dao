# Release Packets — Index

This directory contains release packets generated by the DAO for each executed proposal.

## Naming Convention

```
vX.Y.Z-[slug].md
```

Examples:
- `v0.2.0-performance-optimization-v2.md`
- `v0.3.0-performance-optimization.md`

## Release Packet Contents

Each release packet includes:

| Section | Description |
|---------|-------------|
| Version | Semantic version number |
| Changelog | Summary of changes |
| Release Notes | Full details from deliberation |
| Pre-Release Checklist | Security, risk, test, rollback checks |
| Rollback Plan | How to revert if things go wrong |
| Store Notes | Summary for marketplace/store listing |

## Pre-Release Checklist

- ✅ All control gates passed
- ✅ Risk assessment reviewed
- ✅ Test plan defined
- ✅ Rollback plan documented
- ⬜ Stakeholder sign-off obtained
- ⬜ Monitoring configured
